# AI Boundaries

## Allowed Functions
1. generateBio(cvText)
   - Input: raw CV text only
   - Output: concise employee bio
2. generateContract(employeeData)
   - Input: structured employee data only
   - Output: contract template/PDF-ready content

## Prohibited AI Behavior
- no direct database access
- no compensation access or inference
- no personality analysis
- no scoring/ranking
- no HR advice generation
- no compliance advice generation
